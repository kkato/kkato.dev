### website
This website is created using Hugo (static site generator) and Netlify (hosting service). The source is managed on GitHub and is configured to detect updates to the repository and deploy automatically. I am using [LoveIt](https://themes.gohugo.io/themes/loveit/) for the template and set it up by following the [exampleSite](https://github.com/dillonzq/LoveIt/tree/master/exampleSite).\
